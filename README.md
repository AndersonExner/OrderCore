# OrderCore

OrderCore is a distributed backend system designed to simulate a real-world order processing workflow.

The system focuses on:
- event-driven architecture
- scalability
- resiliency
- clean architecture principles

## Current Tech Stack

- .NET 10
- ASP.NET Core
- Entity Framework Core
- PostgreSQL
- React
- Vite

## Planned

- RabbitMQ
- Redis
- Docker
- Azure
